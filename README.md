# HisoHiso Chat

Simple chat system.


## Description

Used technology is following.

* node.js
* Express4.0
* Socket.IO


## Usage

Run the following command.

    `npm start`


## Install

1. Clone for Github.
2. Run the following command.  <br>
    `npm install`
